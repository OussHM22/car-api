# CAR API REST - Gestion de Voitures Classiques

API REST pour gérer une collection de voitures classiques.  
Développée avec **Node.js**, **Express** et **SQLite**.

## Prérequis

- Node.js (version 16 ou supérieure)
- npm
- Postman (pour tester l’API)
- Git + compte GitHub
